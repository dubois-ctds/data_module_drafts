# DUBOIS CTDS Data Modules

![DUBOIS CTDS logomark](assets/images/logo.png)

Welcome to the **Jupyter Book** for [DUBOIS CTDS](https://dubois-ctds.github.io/) module materials in this repository. Content is grouped by **module** in the sidebar (for example **Fundamentals**).

## What is here

- Lectures: instructional content found in **`<module>/lecture_notebooks/`**
- Labs: practice exercises found in **`<module>/labs/`**
- Homeworks: assignments found in **`<module>/homeworks/`**

## How to use this textbook

The live book is at **[dubois-ctds.github.io/data_module_drafts](https://dubois-ctds.github.io/data_module_drafts/)**. You can read pages statically or run code interactively as follows.

### Launch and run code

**Quick start**

1. **Start the kernel:** Click the **power** button (top right of the page). Wait until the kernel or environment is ready.
2. **Run code:** Use the **play** button next to a cell to run it, or the **Run all** control (top right; it stays visible as you scroll) to run every cell on the page.

:::{note}
**First time on Binder?** The first time you launch with Binder, building the environment can take a few minutes. Later visits are usually faster.
:::

### Run in Binder or your own hub

:::{tip}
**External link** (next to the play button): Opens the notebook in a new browser tab using [Binder](https://mybinder.org/) (free, no account required). The book is wired to the **`dubois-ctds/data_module_drafts`** repository on **`main`**.
:::

:::{note}
**Rocket ship** (launch menu): Choose **Binder** or **Custom** and paste the URL of your own JupyterHub (for example your course or department hub).
:::

:::{seealso}
**Download:** Use the download control to save the notebook (for example `.ipynb`).

**View source:** Use the GitHub icon to open the source file in the repository.
:::

For the broader organization site, see [dubois-ctds.github.io](https://dubois-ctds.github.io/).
